# C++介绍

现代C++语言可以看作三部分组成:

* 低级语言，大部分继承自C语言。
* 现代高级语言特性，允许我们定义自己的类型以及组织大规模程序和系统。
* 标准库，它利用高级特性来提供有用的数据结构和算法。
